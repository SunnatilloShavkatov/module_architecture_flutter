# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-26

### Added
- Initial architectural foundation.
- Core package with Clean Architecture abstractions.
- Modular dependency injection pattern (merge_dependencies).
- Feature module skeletons (auth, home, main, more).
- Standardized UI component library (components).
- Centralized navigation system using GoRouter.
- Detailed architecture documentation and contribution guidelines.
